this is a ticketing system that vendor add tickets and customer can buy. vendor have a separate email and vendor can log in to the system using that specific email and password. customers can signup first and then they can log in to the system. vendor can add tickets with event name, date, price total number of tickets and a description. they edit the details in the tickets or delete it. after vendor add the tickets customers can see the tickets that added by in the customer dashboard. customer can buy maximum 1 to 5 tickets at a time. after select the amount and click on the buy button selected amount will be reduced from the total tickets

Angular version: 18.2.12

to run the application first install spring boot version and angular versions first. run the spring boot backend first and then run the angular front by command ng serve. web app will be run on http://localhost:4200
